CSS Cheat Sheet

CSS Property Description Example
color Changes the color of text color: #333;
background-color Sets the background color of an element background-color: #f4f4f4;
font-family Changes the font family of certain words, sentences, paragraphs, etc. font-family: Arial, sans-serif;
font-size Used to modify the size of the displayed font font-size: 16px;
margin Sets the margin around an element margin: 10px;
padding Sets the padding inside an element padding: 15px;
border Sets the border of an element border: 1px solid #ccc;
display Sets the display type of an element display: value;
text-align Aligns the text inside an element text-align: center;
