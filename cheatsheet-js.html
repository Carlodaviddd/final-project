JavaScript Cheat Sheet

JavaScript Feature Description Example
console.log() Outputs a message to the console console.log('Hello World!');
alert() Displays an alert box with a message alert('Hello!');
var Used to initialize to value, redeclared and its value can be reassigned var y = 10;
let Similar to var but is blocked scope let x = 5;
const Declares a constant value that cannot be changed const pi = 3.14;
function Objects that can be called to execute a block of code function greet() {
alert('Hello!');
}
if/else Conditional statements if (x > 5) {
console.log('Yes');
} else {
console.log('No');
}
for loop Loops a block of code a certain number of times for (let i = 0; i < 5; i++) {
console.log(i);
}
Array Stores multiple values of same type in a single variable let arr = [1, 2, 3];
object Complex data type that allows us to store a collection of data let obj = { name: 'Carlo', age: 25 };
